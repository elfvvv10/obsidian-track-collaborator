# Local LLMs

Running models locally can improve privacy, control, and offline access.

Ollama makes it easier to run local language models behind a simple HTTP API.

For a local RAG system:

- A chat model answers questions from retrieved context.
- An embedding model converts text into vectors for semantic search.
- A vector database stores embeddings for later retrieval.

Local systems are easier to understand when the architecture is modular and the dependencies are minimal.

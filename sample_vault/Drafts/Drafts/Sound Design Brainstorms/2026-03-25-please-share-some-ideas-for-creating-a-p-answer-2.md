## Sound Design Brainstorm

Quick Answer: A progressive house bassline should have a driving, melodic groove with subtle buildups and drops.

Production Recipes:
1. Name: Reese Bass Synth
   Groove / MIDI: Use a repetitive low note pattern with slight variations in length to create tension (e.g., quarter note on beat 4, eighth note rests every 2 bars)
   Sound Design: Create a supersaw synth by detuning multiple saw oscillators and automating the filter cutoff for movement
   How to Build It:
    a. In your DAW, create a new MIDI track and import your low note pattern
    b. Create multiple saw oscillators in your synthesizer (e.g., Serum or Sylenth1) and detune them slightly from each other (up to 2-4 semitones)
    c. Route the oscillators into a new synth and apply unison mode for width
    d. Add a filter and automate the cutoff to add movement
   Where to Use It: This bassline works well in the drop section to create a buildup and release

2. Name: Staccato Sub Bass
   Groove / MIDI: Use short, punchy sub bass hits on every quarter note with occasional longer notes for emphasis
   Sound Design: Create a deep, analog-style sub bass using a low-pass filter and distortion
   How to Build It:
    a. In your DAW, create a new MIDI track and import your staccato sub bass pattern
    b. Choose a synthesizer with a good-quality low-pass filter (e.g., Serum or Massive)
    c. Create a saw oscillator and apply a low-pass filter to shape the sound
    d. Add distortion to the oscillator for warmth and depth
    e. Automate the filter cutoff slightly for movement
   Where to Use It: This bassline works well in the intro and breakdown sections to create tension and build anticipation

Optional Variations:
1. Name: Chord Stab Bass
   Groove / MIDI: Use a repeating chord progression with staccato hits on each note
   Sound Design: Choose a synth with good-preset selection (e.g., Spire or Serum)
   How to Build It:
    a. In your DAW, create a new MIDI track and import your chord stab pattern
    b. Select a preset that fits the harmonic context of your track from your synthesizer's preset library
    c. Adjust the volume of individual notes in the preset to fit your desired rhythm and groove
   Where to Use It: This bassline works well in the build-up sections to create melodic tension and release

2. Name: Funky Drum Fill Bass
   Groove / MIDI: Use a syncopated drum fill pattern with varying lengths and velocities
   Sound Design: Create a tight, punchy snare sound using one-shot samples or a synthesizer
   How to Build It:
    a. In your DAW, create a new MIDI track and import your funky drum fill pattern
    b. Choose a synthesizer with a good snare preset (e.g., Spire or Massive)
    c. Adjust the velocity and length of individual notes in the preset to create the desired groove
   Where to Use It: This bassline works well in the transition sections between breakdowns and drops to add energy and excitement

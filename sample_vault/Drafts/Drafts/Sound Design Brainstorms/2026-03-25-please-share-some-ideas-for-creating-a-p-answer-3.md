## Sound Design Brainstorm

Quick Answer:
Create a progressive house bassline by using a combination of melodic basslines and deep, driving low-end frequencies.

Production Recipes:

1. Name: Classic Progressive House Pluck Bassline

Groove / MIDI:
- Use a piano roll to program a repeating pattern of quarter notes on the grid.
- Accent every other quarter note for added impact.

Sound Design:
- Choose an analogue modelled synthesizer with a plucky, string-like tone.
- Apply a light chorus effect and subtle stereo widening for depth.

How to Build It:
1. Set up your synth in the MIDI track.
2. Record your MIDI pattern directly into the DAW.
3. Turn on your chorus effect and adjust the mix to taste.
4. Pan the left/right channels of the synth for added width.

Where to Use It:
- This bassline is perfect for driving the intro or pre-drop section of a progressive house track, setting up anticipation.

2. Name: Reese Bass Synth Design

Groove / MIDI:
- Program a simple repeating pattern of half and whole notes on the grid.
- Use a repeating sequence with long note holds for added drama.

Sound Design:
- Choose a supersaw synthesizer patch (detuned saw waves) for that classic Reese bass sound.
- Add some subtle filtering to the main synth layer to create movement in the sound.

How to Build It:
1. Select your supersaw oscillator and set up detune and unison settings.
2. Record your MIDI pattern into the DAW using a repeating sequence.
3. Apply low-pass filtering with an LFO modulating the cutoff for added motion.
4. Add a touch of distortion to add warmth and color.

Where to Use It:
- This bassline is ideal for the main drop section, providing driving lows and big Reese-style hits.
